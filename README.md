## Description
This project is a RESTful microservice for customer account management as part of the DevOps Capstone project.

## Build Status
The build status badge above shows the current status of the CI workflow. A green checkmark (✓) indicates a successful build.

## Endpoints
- `GET /accounts` - List all accounts
- `GET /accounts/<id>` - Read a specific account
- `POST /accounts` - Create a new account
- `PUT /accounts/<id>` - Update an existing account
- `DELETE /accounts/<id>` - Delete an account
